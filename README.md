# Hazem-Mohamed-Portfolio
https://ziadhassan157.github.io/Hazem-Mohamed-Portfolio/

A modern, responsive portfolio website designed for **Hazem Mohamed** (Data Engineer).

## Developed By
**Ziad Ahmed**

---

## Features
- **Modern UI:** Clean typography with a focus on data engineering content and projects.
- **Dark/Light Mode:** Seamless theme switching with user preference persistence.
- **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, Desktop).
- **Interactive Animations:** Scroll-reveal effects, smooth transitions, and dynamic project timelines.
- **Premium Icons:** Custom-styled social and contact icons with hover glow effects.
- **SEO Optimized:** Structured semantic HTML for better search engine ranking.

## Tech Stack
- **HTML:** Semantic structure.
- **CSS:** Modern layouts using Flexbox, Grid, and CSS Variables.
- **JavaScript:** Theme logic and interactive components.

## Project Structure
- `index.html`: Core structure and content.
- `style.css`: Comprehensive styling and theme definitions.
- `script.js`: Interactive logic and reveal animations.
